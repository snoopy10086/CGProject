## Keboard Operation

- ws是你视角方向前后移动
- ad是视角方向在水平平面的投影的法线方向移动 就是在同一个水平面里动
- zc就是上下动
- e控制机械爪抓取；
- r控制机械臂1三个自由度；
- y控制机械臂2三个自由度；
- ik控制机械爪上下移动；
- 12345分别控制门窗开关；
- t控制传送带转动（按一下动一下）。
